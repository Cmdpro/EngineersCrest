A mod that adds a new crest that will definetly break your game

Slots: 3x Red, 2x Blue, 2x Yellow
Every red item will have 3 uses however the reds will instead spawn a turret which will use the item on a timer at no cost
Turrets can be killed (equivelant of 8 masks)
If you use a red item that is the same type as a currently alive turret, it will relocate the turret at no cost
Turret attack cooldowns shorten the higher the uses on the red item is
If you hold up while binding, you will heal 3 masks on all turrets instead of yourself
If you hold down while binding, you will overclock all turrets causing them to shoot at 2x speed for 10 seconds
